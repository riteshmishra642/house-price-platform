# Assets

Static assets referenced by documentation (README screenshots, architecture
diagram exports, logos). The architecture/data-flow/model-flow diagrams
themselves live inline in `README.md` as Mermaid code blocks (rendered
natively by GitHub) rather than as image files here, so they stay in sync
with the code without needing manual re-export.

Suggested additions once you have a running instance:
- `dashboard_screenshot.png` — Streamlit dashboard prediction tab
- `api_swagger_screenshot.png` — FastAPI `/docs` page
- `model_comparison_screenshot.png` — leaderboard table/chart
